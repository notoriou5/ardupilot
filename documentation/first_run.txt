	-> sim_vehicle.py -f jsbsim:Winddrone --console --map //< starts the Winddrone without tether simulation

	-> wp load ../Tools/autotest/ArduPlane-Missions/CMAC-turns.txt //< you can also choose different waypoints

	-> mode auto //< to launch the plane
	-> mode 39 //< switches to eight_sphere flight mode
